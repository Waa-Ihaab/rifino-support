Bienvenue sur la page d’assistance de Rifino, l’application conçue pour apprendre la langue rifaine, aussi appelée tarifit, de manière simple, moderne et interactive.

Comment pouvons-nous aider ?

  • Signaler un problème technique
  • Poser une question sur l’application
  • Proposer une amélioration
  • Signaler une erreur de traduction
  • Signaler un problème avec un audio

Informations à fournir

Pour nous aider à résoudre le problème rapidement, indiquez si possible :

  • Le modèle de votre appareil
  • La version d’iOS
  • Une description du problème
  • Une capture d’écran
